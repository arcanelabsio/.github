# .github

Community health + org profile files for [@arcanelabsio](https://github.com/arcanelabsio).

## What's in here

- [`profile/README.md`](profile/README.md) — rendered on [github.com/arcanelabsio](https://github.com/arcanelabsio) as the org's public profile page.

## Editing the org profile

Edit `profile/README.md` and push to `main`. GitHub renders the change on the org profile within a minute.

## Per-repo defaults (future)

This repo can also hold org-wide defaults that downstream repos inherit unless they override:

- `ISSUE_TEMPLATE/` — default issue templates
- `PULL_REQUEST_TEMPLATE.md` — default PR template
- `CODE_OF_CONDUCT.md` — default code of conduct
- `CONTRIBUTING.md` — default contributing guide
- `SECURITY.md` — default security policy
- `FUNDING.yml` — default funding config
- `workflows/` — reusable GitHub Actions workflows

See [GitHub's docs on community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for precedence rules.
