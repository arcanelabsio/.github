# Getting Help

Thanks for using an Arcane Labs project. Here's how to find the right support channel.

## Product support

### Longeviti (app & framework)

- **App bugs or feedback**: open an issue on [`longeviti-framework`](https://github.com/arcanelabsio/longeviti-framework/issues) (the app source is private; the framework repo tracks app-facing issues)
- **Documentation**: [longeviti.app/docs](https://longeviti.app/docs/)
- **Framework questions**: [`longeviti-framework` discussions](https://github.com/arcanelabsio/longeviti-framework/discussions) (if enabled) or issues

### Vael

- **Access request** (source currently private): email **support@arcanelabs.info**
- **Product feedback**: email **support@arcanelabs.info**

### Forge

- **Issues & bugs**: [`forge` issue tracker](https://github.com/arcanelabsio/forge/issues)
- **CLI docs**: see the repo README and `docs/` folder
- **npm package**: [`forge-ai-assist`](https://www.npmjs.com/package/forge-ai-assist)

### drive_sync_flutter

- **Issues & bugs**: [issue tracker](https://github.com/arcanelabsio/drive_sync_flutter/issues)
- **Package**: [pub.dev/packages/drive_sync_flutter](https://pub.dev/packages/drive_sync_flutter)

### Shellcraft

- **Issues & bugs**: [issue tracker](https://github.com/arcanelabsio/shellcraft/issues)

## What to include in a bug report

- Version of the tool/app
- Operating system and version
- What you expected vs what happened
- Steps to reproduce (minimum set)
- Relevant logs (redact anything sensitive)

## Security issues

**Do not open a public issue for security vulnerabilities.** Email **security@arcanelabs.info**. See [SECURITY.md](./SECURITY.md) for the full policy.

## Contact directory

- **Say hello / general inquiries**: hello@arcanelabs.info
- **Product support**: support@arcanelabs.info
- **Security vulnerabilities**: security@arcanelabs.info
- **Privacy / data requests**: privacy@arcanelabs.info
- **Community & Code of Conduct**: community@arcanelabs.info
- **Partnerships, business, trademark**: contact@arcanelabs.info
- **Press & media**: social-apps@arcanelabs.info

Our full contact directory with purposes is at [arcanelabs.info/contact](https://arcanelabs.info/contact/).

## Response times

We're a small team and support is best-effort. Typical response windows:

- **Security reports**: 3 business days
- **Bug reports**: 5-7 business days
- **Feature requests**: no SLA — we track everything and revisit during planning
